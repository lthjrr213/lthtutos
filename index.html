HTML

HyperText Markup Language — la structure et le contenu de chaque page web. Créé par Tim Berners-Lee en 1991.

5
Version actuelle
140+
Balises
1991
Créé en
📄
Balises (Tags)
Les balises définissent les éléments. Elles s'ouvrent <tag> et se ferment </tag>. Certaines sont auto-fermantes comme <img /> ou <br />.
🔖
Attributs
Les attributs fournissent des informations supplémentaires aux balises. Ex: <a href="url">, <img src="photo.jpg" alt="desc">
🌳
DOM — Arbre
Le navigateur parse le HTML et construit un arbre d'objets appelé DOM (Document Object Model) que JS peut manipuler.
♿
Accessibilité
HTML sémantique (<nav>, <main>, <article>) aide les lecteurs d'écran. Les attributs ARIA complètent l'accessibilité.
🔍
SEO
Les balises <title>, <meta>, <h1>-<h6> et la structure sémantique influencent directement le référencement.
📱
HTML5
La version actuelle apporte <video>, <audio>, <canvas>, <section>, <article> et l'API localStorage.

CSS

Cascading Style Sheets — la mise en forme visuelle des pages web. Variables, Flexbox, Grid, Animations.

🎨
Cascade & Spécificité
Les styles "cascadent" selon leur spécificité : inline > id > class > tag. Le dernier déclaré gagne à égalité.
📦
Box Model
Chaque élément est une boîte : content → padding → border → margin. box-sizing: border-box inclut padding+border dans la taille.
🧮
Variables CSS
--ma-variable: valeur; dans :root. Utilisation avec var(--ma-variable). Idéal pour les thèmes et la cohérence.
📐
Flexbox
Disposition unidimensionnelle (ligne ou colonne). display: flex. Align-items, justify-content, flex-wrap, gap.
🔲
CSS Grid
Disposition bidimensionnelle (lignes ET colonnes). display: grid. grid-template-columns, grid-area, gap.
✨
Animations
@keyframes pour définir une animation. animation: nom durée easing. transition pour les changements d'état au hover.

JavaScript

Le seul langage de programmation natif du navigateur. Interactivité, logique, animations, APIs et bien plus.

ES14
Version actuelle
#1
Langage populaire
1995
Créé en
⚡
Variables — let / const
const pour les valeurs fixes, let pour les valeurs changeantes. Éviter var (portée problématique et hoisting surprenant).
🔧
Fonctions Arrow
const fn = (a, b) => a + b; Syntaxe concise ES6. Pas de propre "this" — hérite du contexte parent.
🔄
Event Loop
JS est mono-thread mais non-bloquant. L'Event Loop gère la file des callbacks asynchrones (setTimeout, fetch…).
🔒
Closure
Une fonction "se souvient" des variables de son contexte de création, même quand ce contexte n'est plus actif.
🌐
DOM & Events
querySelector, addEventListener, innerHTML. JavaScript modifie le HTML en temps réel via le Document Object Model.
⏳
Promises & Async
async/await rend le code asynchrone aussi lisible que du code synchrone. try/catch pour la gestion d'erreurs.

JavaScript — Fonctions 
// Déclaration (hoistée)
function saluer(nom = "visiteur") {
  return `Bonjour, ${nom}!`;
}

// Expression (arrow function)
const carre = x => x * x;
const somme = (a, b) => a + b;

// Rest parameters
const addTout = (...nums) =>
  nums.reduce((acc, n) => acc + n, 0);
addTout(1, 2, 3, 4); // 10

// Méthodes de tableau
const nums = [1, 2, 3, 4, 5];

nums.map(x => x * 2);          // [2,4,6,8,10]
nums.filter(x => x % 2 === 0); // [2,4]
nums.reduce((a, x) => a + x, 0); // 15
nums.find(x => x > 3);          // 4

// Closure
function compteur() {
  let n = 0;
  return {
    inc: () => ++n,
    val: () => n
  };
}
const c = compteur();
c.inc(); c.inc(); c.val(); // 2

// Classe ES6
class Animal {
  constructor(nom) { this.nom = nom; }
  parler() { return `${this.nom} fait du bruit`; }
}

JavaScript — Async / Await & Fetch 
// Promise
const maPromesse = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Succès!"), 1000);
});

maPromesse
  .then(data => console.log(data))
  .catch(err => console.error(err));

// async / await — bien plus lisible!
async function chargerUtilisateur(id) {
  try {
    const response = await fetch(`/api/users/${id}`);

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}`);
    }

    const user = await response.json();
    return user;

  } catch (err) {
    console.error("Erreur:", err.message);
    throw err;
  }
}

// POST avec fetch
async function creerUser(data) {
  const res = await fetch("/api/users", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(data)
  });
  return res.json();
}

// Promise.all — parallèle
const [users, posts] = await Promise.all([
  fetch("/api/users").then(r => r.json()),
  fetch("/api/posts").then(r => r.json())
]);

Exercice 01

To-Do List

Crée une application de liste de tâches complète avec HTML, CSS et JavaScript.

Fonctionnalités requises
  • ✦ Ajouter une tâche via un champ texte
  • ✦ Cocher une tâche comme terminée
  • ✦ Supprimer une tâche
  • ✦ Afficher le nombre de tâches restantes
HTML CSS JavaScript
Solution complète
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8"/>
  <title>To-Do List</title>
  <style>
    * { box-sizing: border-box; margin: 0; padding: 0; }
    body { font-family: sans-serif; background: #0f0f1a;
           color: #f0f0f8; display: flex; justify-content: center;
           align-items: flex-start; min-height: 100vh; padding: 2rem; }
    .app { background: #1a1a28; border-radius: 16px;
             padding: 2rem; width: 100%; max-width: 480px;
             box-shadow: 0 8px 32px rgba(0,0,0,0.4); }
    h1 { font-size: 1.5rem; margin-bottom: 1.5rem;
          color: #f7ca18; }
    .input-row { display: flex; gap: 0.5rem; margin-bottom: 1.25rem; }
    input { flex: 1; padding: 0.6rem 1rem; border-radius: 8px;
             border: 1px solid #2a2a3a; background: #0a0a0f;
             color: #f0f0f8; font-size: 0.95rem; }
    button.add { padding: 0.6rem 1.2rem; border-radius: 8px;
                   background: #f7ca18; color: #000; border: none;
                   font-weight: 700; cursor: pointer; }
    ul { list-style: none; display: flex; flex-direction: column; gap: 0.5rem; }
    li { display: flex; align-items: center; gap: 0.75rem;
          padding: 0.75rem 1rem; background: #0f0f1a;
          border-radius: 10px; border: 1px solid #2a2a3a; }
    li.done span.text { text-decoration: line-through; color: #666; }
    span.text { flex: 1; }
    button.del { background: rgba(227,76,38,0.15); color: #e34c26;
                    border: 1px solid rgba(227,76,38,0.3); border-radius: 6px;
                    padding: 2px 10px; cursor: pointer; font-size: 0.8rem; }
    .count { margin-top: 1rem; font-size: 0.85rem; color: #8888aa; }
  </style>
</head>
<body>
  <div class="app">
    <h1>📝 Ma To-Do List</h1>
    <div class="input-row">
      <input type="text" id="taskInput" placeholder="Nouvelle tâche..."/>
      <button class="add" onclick="addTask()">Ajouter</button>
    </div>
    <ul id="taskList"></ul>
    <p class="count" id="count">0 tâche restante</p>
  </div>
  <script>
    const input = document.getElementById('taskInput');
    const list  = document.getElementById('taskList');
    const count = document.getElementById('count');
    let tasks = [];

    input.addEventListener('keydown', e => { if (e.key === 'Enter') addTask(); });

    function addTask() {
      const txt = input.value.trim();
      if (!txt) return;
      tasks.push({ text: txt, done: false });
      input.value = '';
      render();
    }
    function toggle(i) { tasks[i].done = !tasks[i].done; render(); }
    function remove(i) { tasks.splice(i, 1); render(); }
    function render() {
      list.innerHTML = tasks.map((t, i) => `
        <li class="${t.done ? 'done' : ''}">
          <input type="checkbox" ${t.done ? 'checked' : ''}
                 onchange="toggle(${i})">
          <span class="text">${t.text}</span>
          <button class="del" onclick="remove(${i})">✕</button>
        </li>`).join('');
      const rem = tasks.filter(t => !t.done).length;
      count.textContent = rem + (rem <= 1 ? ' tâche restante' : ' tâches restantes');
    }
  </script>
</body></html>

<body>
  <div class="calc">
    <div class="screen">
      <div class="expr" id="expr"></div>
      <div class="result" id="result">0</div>
    </div>
    <div class="grid">
      <button class="btn-clr" onclick="clear()">C</button>
      <button class="btn-op"  onclick="press('/')">÷</button>
      <button class="btn-op"  onclick="press('*')">×</button>
      <button class="btn-num" onclick="press('7')">7</button>
      <button class="btn-num" onclick="press('8')">8</button>
      <button class="btn-num" onclick="press('9')">9</button>

      <button class="btn-num" onclick="press('4')">4</button>
      <button class="btn-num" onclick="press('5')">5</button>
      <button class="btn-num" onclick="press('6')">6</button>

    let expr = '';

      expr += v;
      document.getElementById('expr').textContent = expr;

      try {
        const res = Function('"use strict"; return (' + expr + ')')();

        document.getElementById('result').textContent = 'Erreur';
        expr = '';
      }

      expr = '';
      document.getElementById('expr').textContent = '';
      document.getElementById('result').textContent = '0';
    }
  </script>
